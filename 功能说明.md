# 功能说明

## 📸 OCR（光学字符识别）

### 是什么？
**OCR (Optical Character Recognition)** 是光学字符识别技术，可以从图片中提取文字。

### 在这个应用中做什么？
- 📷 **识别图片中的文字**：上传或选择一张包含文字的图片
- ✍️ **提取文本**：自动识别图片中的所有文字
- 📝 **转换为可编辑文本**：将图片中的文字提取出来，可以复制、编辑或用于其他功能

### 使用场景：
- 📚 扫描课本或笔记，提取文字内容
- 📄 识别文档、PDF 中的文字
- 🖼️ 从截图或照片中提取文字
- 📖 将纸质材料数字化

### 技术实现：
- 使用 macOS 的 **Vision 框架**
- 支持多语言识别（默认：英文）
- 自动清理和格式化提取的文字

## 🃏 Flashcards（闪卡）

### 是什么？
**Flashcards（闪卡）** 是一种学习工具，每张卡片包含一个问题和一个答案，用于记忆和复习。

### 在这个应用中做什么？
- 🤖 **自动生成闪卡**：从文本内容自动生成学习卡片
- 📚 **问题-答案对**：每张卡片包含一个问题和一个答案
- 🧠 **辅助记忆**：通过反复查看问题并回忆答案来加强记忆

### 使用场景：
- 📖 从学习材料（如 OCR 提取的文字）生成闪卡
- 🎓 准备考试，创建复习卡片
- 📝 将长文本转换为易记忆的问答形式
- 🔄 定期复习重要概念

### 工作流程：
1. 提供文本内容（可以是 OCR 提取的文字，或直接输入）
2. AI 自动分析内容，提取关键概念
3. 生成多个问题-答案对（至少 3 张卡片）
4. 可以查看、复习这些卡片

### 示例：
```
问题：什么是 SwiftUI？
答案：SwiftUI 是 Apple 的声明式 UI 框架，用于构建 iOS、macOS 等应用界面。

问题：什么是 MVVM 架构？
答案：MVVM 是 Model-View-ViewModel 架构模式，用于分离业务逻辑和 UI。
```

## 🍅 Pomodoro（番茄工作法）

### 是什么？
**Pomodoro（番茄工作法）** 是一种时间管理技巧，将工作时间分成 25 分钟的专注时段，每个时段后休息 5 分钟。

### 在这个应用中做什么？
- ⏱️ **25 分钟计时器**：默认专注时长为 25 分钟（1500 秒）
- ▶️ **开始/暂停**：可以随时开始或暂停计时
- 🔄 **重置**：重置到初始 25 分钟
- 📊 **专注时间管理**：帮助你保持专注，提高学习效率

### 使用场景：
- 📚 **专注学习**：设置 25 分钟专注学习时间
- 💻 **工作时段**：将工作分成多个 25 分钟时段
- 🎯 **提高效率**：通过定时器保持专注，避免分心
- ⏰ **时间感知**：更好地感知和管理时间

### 工作流程：
1. 点击 "Pomodoro" 按钮打开计时器
2. 点击 "Start" 开始 25 分钟倒计时
3. 专注工作/学习，直到计时器结束
4. 休息 5 分钟（可以手动设置）
5. 重复循环

### 番茄工作法原理：
- 🍅 **25 分钟专注**：短时间专注，更容易保持注意力
- ☕ **5 分钟休息**：短暂休息，恢复精力
- 🔁 **循环重复**：每 4 个番茄时段后，可以休息更长时间（15-30 分钟）

## 🎯 三个功能的协同使用

这三个功能可以组合使用，形成完整的学习工作流：

### 典型学习流程：

1. **📸 OCR 提取文字**
   - 扫描课本或笔记
   - 提取文字内容

2. **🃏 生成闪卡**
   - 将 OCR 提取的文字输入到闪卡生成器
   - AI 自动生成学习卡片

3. **🍅 使用 Pomodoro 复习**
   - 打开 Pomodoro 计时器
   - 在 25 分钟内专注复习闪卡
   - 休息后继续下一轮

### 完整示例：

```
1. 用 OCR 扫描课本第 5 页
   ↓
2. 提取文字："Swift 是一种编程语言..."
   ↓
3. 生成闪卡：
   - 问题：什么是 Swift？
   - 答案：Swift 是一种编程语言...
   ↓
4. 打开 Pomodoro，设置 25 分钟
   ↓
5. 在 25 分钟内专注复习这些闪卡
   ↓
6. 休息 5 分钟，然后继续
```
